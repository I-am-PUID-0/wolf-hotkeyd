# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-07-02

### Features

- Initial controller hotkey daemon for Wolf Steam containers.
- Added dry-run and action execution modes.
- Added force-close action helpers and capture scripts.
- Added custom Steam runner image scaffold.
- Added host-side action mode for reduced resident tooling inside game
  containers.
